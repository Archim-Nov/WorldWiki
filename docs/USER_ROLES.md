# 用户分级清单

本项目目前只区分两类用户：**普通用户** 与 **可编辑用户**。

## 普通用户
权限范围：
- 浏览所有公开页面
- 查看详情页与相关推荐
- 无法进入 Studio（编辑后台）
- 无法创建/编辑内容

适用人群：
- 普通读者
- 仅浏览或反馈的用户

## 可编辑用户
权限范围：
- 拥有普通用户所有权限
- 可登录并进入 Profile（/dashboard）
- 通过 Profile 进入 Studio（/studio）
- 在 Studio 内创建与编辑内容
- 可发布内容（是否允许发布取决于实际账号权限）

适用人群：
- 编辑与内容运营
- 管理员或被授权的贡献者

## 授权与管理（现阶段）
- 账号体系由 Supabase Auth 管理
- 可编辑用户需由管理员分配权限（Studio 账号权限）
- 未登录用户访问 /studio 会被引导至 /login

## 后续可扩展方向（可选）
- 审核者/管理员角色
- 贡献者编辑但需审核发布
- 页面保护（高风险页面仅可编辑用户可改）
